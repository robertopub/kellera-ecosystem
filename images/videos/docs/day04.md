# Day 04 — RGB + Temperature System

Today the Kellera project achieved a major milestone.

The ESP32 is now capable of:
- reading temperature in real time
- processing the information
- responding visually using RGB colors

---

# Components

- ESP32 DevKit V1
- DS18B20 temperature sensor
- RGB LED HW-479
- Protoboard
- Jumper wires

---

# System Response

🔵 Cold temperature  
🟢 Normal temperature  
🔴 High temperature
