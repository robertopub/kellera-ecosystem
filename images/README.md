# Dia 04 — Sistema RGB + Temperatura

Hoje o projeto Kellera atingiu um marco importante.

A ESP32 agora consegue:
- ler temperatura em tempo real
- processar os dados
- responder visualmente usando cores RGB

---

# 🔧 Componentes Utilizados

- ESP32 DevKit V1
- Sensor de temperatura DS18B20
- LED RGB HW-479
- Protoboard
- Jumpers

---

# 🌈 Resposta do Sistema

🔵 Temperatura baixa

🟢 Temperatura normal

🔴 Temperatura alta
